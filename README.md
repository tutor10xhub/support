# Tutor10x Support and Release Notes

Welcome to the official support and release notes repository for Tutor10x! Here, you'll find:

- Detailed **release notes** for Tutor10x products, including Lab Booster and AutoPilot.
- A platform to **report bugs**, **request features**, and share **workarounds**.
- **Known issues** and solutions contributed by the community.

## How to Use This Repository

- **Submit an Issue**: Use the [Issues tab](https://github.com/tutor10xhub/support/issues) to report a bug, request a feature, or ask a question.
- **Explore Release Notes**: Visit the `/releases/` directory for detailed updates.
- **Get Involved**: Join our community by contributing solutions, enhancements, and feedback.

## Support Options

- Free users can access this repository for community-driven support.
- Paid users can access priority support via [Tutor10x Premium Support](https://tutor10x.com/helpdesk).
